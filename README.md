Простой набор тестов для Kontur.Staff на Selenium

Для корректной работы тестов необходимо настроить переменные окружения для авторизации.

1. В корне проекта создайте файл **`.env`** (на основе приложенного `.env.example`).
2. Добавьте в него свои учетные данные:
   STAFF_LOGIN=ваш_логин
   STAFF_PASSWORD=ваш_пароль
